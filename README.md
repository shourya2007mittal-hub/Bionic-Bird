# Bionic-Bird

<img width="862" height="635" alt="Screenshot 2026-06-25 214513" src="https://github.com/user-attachments/assets/2bd5b182-722b-4af7-993c-37f3c956dea0" />



I made a bionic butterfly based on a custom ESP32 Board that holds an IMU and a Barometer for control. Is has 2 Servos each servos control a Wing. It uses a 3.7V battery and one can control it through your phone via bluetooth.

## Features

- Base on ESP32 Module
- Has IMU (MPU_6050)
- Has a Barometer (BMP280)
- Uses 2 High torque Servoes
- Would have LED strips on the wings
